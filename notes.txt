A callback function is a function passed as an argument to another function, and it’s executed after some operation is completed.
You are telling a function, “Hey, once you're done, call this other function I gave you.”
Node.js is asynchronous and non-blocking — it doesn’t wait for tasks like file reading, DB querying, or HTTP requests to finish.
Instead, it uses callbacks to:
Start a task. Do other things in the meantime. Then come back and run the callback once the task finishes
